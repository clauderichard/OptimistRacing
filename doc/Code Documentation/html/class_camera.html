Detailed Description

This class is meant to store the position of a camera that doesn't interact physically with the world. Can store position while frozen in space.

Constructor & Destructor Documentation

Basic constructor, begins unfrozen, at the origin, with a standard orientation.

Member Function Documentation

void Camera::freeze ( )

Freezes the position of the camera, but not its orientation, so that if the point of focus moves, the camera doesn't move with it anymore.

virtual void Camera::glApply ( )
pure virtual

Sets up the camera in OpenGL according to the camera's position and orientation. I.e. multiplies the transformation matrix in OpenGL by the right translations and rotations. You should usually call this in ModelView mode before drawing anything.

Implemented in RacerCamera.

void Camera::unfreeze ( )

Undoes the effect of the freeze method, so that the camera moves as usual again.
