Detailed Description

template<typename T>
class Grid2D< T >

A 2d grid where each cell has a value.

Represents a 2d grid with constant spacing (may be different for x and y) where each cell has a value.

Basic constructor.

In this constructor, everything about the grid is specified in the arguments.

Precondition
The values in the values array must be ordered as follows: (x1,y1), (x2,y1), (x3,y1), ... , (xm,yn).
Postcondition
This object will represent a 2d grid, with origin (originX,originY), x-spacing gridHX, y-spacing gridHY, with numCellsX cells along the x-axis and numCellsY cells along the y-axis. The origin is the bottom-left corner of the grid. This object will not hold a pointer to values; it simply copies the array.
Parameters
originXThe origin's x-value, which is the smallest x-value on the grid.
originYThe origin's y-value, which is the smallest y-value on the grid.
numCellsXThe number of cells counting along the x-axis.
numCellsYThe number of cells counting along the y-axis.
Exceptions
Anerror if values has length smaller than numCellsX * numCellsY. An error if any of gridHX, gridHY, numCellsX, or numCellsY is <= 0.
