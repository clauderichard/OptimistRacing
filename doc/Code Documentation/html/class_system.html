Detailed Description

A physical system in Lagrangian Mechanics, which uses generalized coordinates.

This class handles the computation of given the generalized position vector , the generalized velocity vector , and the generalized applied force vector . In order to use this class you need to implement the computeAB method, which computes the matrix and the vector given and , such that the matrix form of the Lagrange equations can be written .
